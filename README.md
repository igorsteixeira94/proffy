## Como Usar

🐋️ *via docker*

```shell
docker-composer up
```

 👨‍💻️ *sem docker*

```shell
git clone git@github.com:igorsteixeira94/proffy.git
cd proffy

#Iniciando o Servidor localhost:3333
cd server
yarn install
yarn start

#Iniciando o Web localhost:3000
cd web
yarn install
yarn start
```
